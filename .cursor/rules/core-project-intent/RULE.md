──────────────────────────────────────────────
CORE PROJECT INTENT
──────────────────────────────────────────────

This project exists solely for discussion, reasoning, and question-and-answer.

It is NOT a software repository and NOT intended for:
- Writing production code
- Managing files or folders
- Running, building, or deploying anything

All interactions must be treated as technical conversations,
design discussions, or conceptual explorations.

──────────────────────────────────────────────
ABSOLUTE CONSTRAINTS
──────────────────────────────────────────────

DO NOT:
- Create, edit, refactor, or suggest creating files or directories
- Assume any code should be executed
- Assume this project contains or will contain runnable software
- Generate boilerplate, scaffolding, or setup instructions
- Suggest commands, builds, deployments, or environment setup
- Treat this workspace as a typical engineering repository

Any exception to the above is allowed ONLY if the user explicitly asks.

──────────────────────────────────────────────
CODE USAGE POLICY
──────────────────────────────────────────────

Code is NOT the default output.

Only provide code if:
- The user explicitly asks for code, OR
- A minimal illustrative example is strictly necessary
  to explain a concept

When providing code:
- Keep it minimal and non-production
- Prefer pseudocode or simplified examples
- Clearly state that the code is illustrative only
- Always explain intent and reasoning
- Do NOT assume language, framework, or tooling unless specified
